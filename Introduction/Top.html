Pythonのpygameで作成した3Dの空間と、 pandasやnumpyを用いた食事管理のGUIアプリを組み合わせたものです。

2D用のライブラリですが、3Dの座標を2Dに変換するクラスなどを自作して 3Dの描画を行いました。

ここでは、操作方法やプレイ動画、ソースコードの概要を示します。

作成した環境:

  • Python 3.11.1

使用した外部ライブラリ:

  • pygame 2.4.0
  • pandas 1.5.2
  • numpy 1.23.5
  • matplotlib 3.6.2
